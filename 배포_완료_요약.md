### 2️⃣ 클라우드 배포 (선택)

#### 🌟 Render (추천)

**장점:**
- 무료 플랜 제공
- GitHub 자동 연동
- 간단한 설정
- 자동 HTTPS

**배포 방법:**
1. https://render.com 접속
2. GitHub 로그인
3. "New +" → "Web Service"
4. 저장소: `kwkimd/community_crawling`
5. 환경 변수 설정:
   ```
   GEMINI_API_KEY=your_api_key_here
   GEMINI_MODEL=gemini-2.0-flash
   ```
6. "Create Web Service" 클릭

**예상 소요 시간:** 5분

---

#### ⚡ Railway

**장점:**
- 매우 간단한 설정
- 자동 배포
- 무료 플랜

**배포 방법:**
1. https://railway.app 접속
2. GitHub 로그인
3. "New Project" → "Deploy from GitHub repo"
4. 저장소 선택
5. 환경 변수 추가
6. 자동 배포 시작

**예상 소요 시간:** 3분

---

#### 🐳 Docker (고급)

**로컬에서 테스트:**

docker build -t community-dashboard .

## ⚠️ 주의사항

### 데이터베이스
- 클라우드 배포 시 SQLite는 **휘발성**입니다
- 서버 재시작 시 데이터가 **초기화**됩니다
- 영구 저장이 필요하면 PostgreSQL 사용 권장

### API 키 보안
- `.env` 파일은 **절대 커밋 금지**
- 환경 변수로만 API 키 설정
- `.gitignore`에 `.env` 포함 확인 완료 ✅

### 비용
- Render 무료 플랜: 월 750시간 (충분함)
- Railway 무료 플랜: 월 $5 크레딧
- 무료 플랫폼은 비활성 시 슬립 모드 진입 가능
